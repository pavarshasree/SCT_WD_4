# Task 04 - To-Do Web App

## Overview

Task 04 is an advanced To-Do Web Application developed as part of the SkillCraft Technology Web Development Internship. The application helps users efficiently manage daily tasks by allowing them to add, edit, delete, and track tasks with a clean and responsive user interface.

## Features

* Add new tasks
* Edit existing tasks
* Delete tasks
* Mark tasks as completed
* Filter tasks (All, Active, Completed)
* Assign task categories
* Set due date and time
* Task statistics dashboard
* Local Storage support for data persistence
* Responsive design for desktop and mobile devices

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Local Storage API
